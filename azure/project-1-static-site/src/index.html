My Philosophy

I believe in the "Infrastructure as Code" (IaC) methodology. Clicks in a web portal don't scale, and they aren't repeatable. Code is.

This entire website is a live demonstration of that. It's not just a page; it's a fully automated CI/CD pipeline. When I push a change to the GitHub repository, a GitHub Actions workflow triggers, runs my Terraform code to validate the infrastructure, and securely deploys the new static files to Azure Blob Storage. No manual intervention required.
